# Tip Calculator

This is a simple Tip Calculator web application built using HTML, CSS, and JavaScript. It allows users to enter the bill amount and tip percentage to calculate the tip and total bill instantly.

## Features
- Enter bill amount
- Enter tip percentage
- Calculates tip amount automatically
- Shows total bill including tip

## Technologies Used
- HTML
- CSS
- JavaScript

## Learning
This project helped me practice basic JavaScript concepts such as user input, calculations, and DOM manipulation. I also used AI tools to help solve some errors while building this project.
